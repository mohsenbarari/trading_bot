# WebApp UI/UX Redesign V2 — Stage 0A

مرجع اصلی بازبینی، فایل `visual-directions.html` است. این فایل سه جهت موبایل را با سناریو و محتوای یکسان، جدول مقایسه و یک نمونه adaptive دسکتاپ نمایش می‌دهد.

برای بازتولید تصاویر پس از نصب وابستگی‌های frontend و مرورگر Playwright:

```bash
node docs/uiux-stage0/capture-prototype.cjs
```

فایل‌های این پوشه prototype و مدرک تصمیم طراحی‌اند؛ هیچ‌کدام در runtime وب‌اپ بارگذاری نمی‌شوند.
